STEP 1: Install Dependencies
-----------------------------

# Install MongoDB (if not already installed)
sudo apt update
sudo apt install mongodb

# For Django backend
cd server
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# For Flask sentiment service
cd ../microservices/sentiment-analyzer
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# For Node.js car service
cd ../car-inventory
npm install

STEP 2: Initialize Database
----------------------------

# Start MongoDB
sudo systemctl start mongodb
sudo systemctl status mongodb

# Run the initialization script
cd ../../server/database
./init_mongodb.sh

# Or manually initialize with Python
cd ..
python manage.py shell
>>> from djangoapp.restapis import init_sample_data

STEP 3: Run Django Migrations
------------------------------
cd server
source venv/bin/activate
python manage.py makemigrations
python manage.py migrate

# Create superuser for admin panel
python manage.py createsuperuser
# Username: admin (or your choice)
# Password: (choose a secure password)

STEP 4: Start All Services
---------------------------

# Terminal 1 - Django Backend
cd server
source venv/bin/activate
python manage.py runserver
# Runs on: http://localhost:8000

===============================================================================
TROUBLESHOOTING
===============================================================================

MongoDB Connection Issues:
- Check if MongoDB is running: sudo systemctl status mongodb
- Verify connection string in restapis.py
- Try: mongosh to connect manually

Django Errors:
- Make sure virtual environment is activated
- Run migrations: python manage.py migrate
- Check INSTALLED_APPS in settings.py

Port Already in Use:
- Find process: lsof -i :8000
- Kill process: kill -9 <PID>

CORS Errors:
- Verify django-cors-headers is installed
- Check CORS_ALLOW_ALL_ORIGINS = True in settings.py
